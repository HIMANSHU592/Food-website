Project Overview

The Food Ordering Website is a frontend-based web application that allows users to browse food items, add them to a cart, and place orders easily. The project focuses on creating a responsive and user-friendly interface while implementing core functionalities like cart management and form validation.

🚀 Features
📱 Responsive design (Mobile + Desktop)
🛒 Add items to cart
❌ Remove items from cart
🔢 Dynamic quantity & total calculation
🧾 Order form with input validation
✅ Order confirmation system
🔐 Login page validation (frontend)
🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and responsiveness
JavaScript – Functionality and logic
